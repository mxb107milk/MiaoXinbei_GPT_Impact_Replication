# GPT 時代下各行業研究者論文發表的影響研究——數據集

## 數據集簡介
本數據集為論文《GPT 時代下各行業研究者論文發表的影響研究 —— 基於 5 個領域專屬問題的實證分析與可視化實現》的配套數據與代碼。

研究以 2022 年 11 月 GPT-3.5 正式發布為準自然實驗，基於 OpenAlex 全球開放學術數據庫，選取教育大數據、新媒體傳播、文旅產業、電子商務、人力資源管理 5 個文本密集型領域 2019—2024 年論文元數據為樣本（共 30,000 篇，每領域 6,000 篇），通過獨立樣本 t 檢驗、雙向固定效應面板回歸及異質性分析，系統檢驗 GPT 發布對學術論文在創作效率、文本創新、合作特徵、規範邊界四個維度的影響。

## 數據來源
- **數據源**: OpenAlex 全球開放學術數據庫 (https://openalex.org/)
- **採集時間**: 2025年1月
- **時間範圍**: 2019年1月1日 — 2024年12月31日
- **樣本規模**: 30,000篇論文（5個領域各6,000篇）
- **核心變量**: 月度論文產出量、關鍵詞新穎度、摘要原創度、跨機構合作率、跨國合作率、參考文獻數量、總引用量、實證方法規範度

## 代碼功能
- 數據採集（OpenAlex API）
- 數據清洗與變量構建（Jaccard 新穎度、文本相似度原創度等）
- 統計檢驗（Welch t 檢驗、分組 t 檢驗）
- 面板回歸（雙向固定效應 + 時間趨勢）
- 可視化（雷達圖、熱力圖、變化趨勢圖、回歸係數圖）
